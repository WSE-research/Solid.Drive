= constants

All hardcoded values for the app, organized by domain.
When you find yourself writing a literal string or number that has meaning beyond the current line, add it here.

== Groups

[cols="1,3", options="header"]
|===
| Group | Constants

| Branding | `APP_NAME`
| Auth | `SOLID_PROVIDERS`, `CUSTOM_PROVIDER_VALUE`
| External links | `EXTERNAL_LINKS`
| i18n | `SUPPORTED_LANGUAGES`, `SUPPORTED_LANGUAGE_CODES`, `FALLBACK_LANGUAGE`, `DEFAULT_LOCALE`, `LANGUAGE_DETECTION_ORDER`, `LANGUAGE_CACHE_LOCATIONS`
| Storage | `APP_CONTAINER_PATH`, `SYSTEM_FILES`, `INDEX_FILE`, `SHARED_CATALOG_PREFIX`, `DEFAULT_CATALOG_FILENAME`, `AVATAR_UPLOAD_PATH`
| File types | `DEFAULT_FILE_TYPE_URI`, `DEFAULT_FILE_TYPES`, `FILE_TYPE_DESCRIPTION_MAX_LENGTH`
| RDF namespaces | `RDF_NAMESPACES`, `RDF_TYPE_URI`
| MIME types | `MIME_PREFIXES`, `CONTENT_TYPES`, `SPREADSHEET_MIME_TYPES`, `DOCUMENT_MIME_TYPES`
| UI | `MAX_DISPLAY_NAME_LENGTH`, `DATE_FORMAT_OPTIONS`, `SHORT_DATE_FORMAT_OPTIONS`, `STORAGE_RETRY_DELAY_MS`
| Validation | `DEFAULT_TBOX_PATH`
|===
