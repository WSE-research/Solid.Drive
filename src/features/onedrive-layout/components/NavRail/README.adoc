= NavRail

Left navigation pane for the OneDrive layout. Two states driven by `useNavRailExpanded`:

Expanded:: Wide pane. Gradient `+ Create or upload` pill, account header with the user's name and a collapse toggle, then five view rows (Home, My Files, Shared, Requests, People) as icon + label. Requests and People sit under a foldable "Browse files by" header.

Collapsed:: Icon-only rail. `+` button, expand toggle, one icon per view. Each icon has a tooltip.

State persists in `localStorage`.
Icons keep the same x-position in both states; only the labels appear or fade.
At `max-width: 960px` the rail is a fixed overlay anchored under the topbar.

== Props

[cols="1,3", options="header"]
|===
| Prop | Description

| `onNewFolder` | "New folder" handler. When omitted (together with `onFilesPicked`) the rail renders a disabled `+` button.
| `onFilesPicked` | "Upload files" handler.
| `accountName` | Display name shown in the account header. Hidden when collapsed.
|===
