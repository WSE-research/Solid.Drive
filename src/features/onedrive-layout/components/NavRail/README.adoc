= NavRail

Left edge navigation rail for the OneDrive layout.
Renders the Create button at the top, three view items in the middle for Home, My Files, and Shared, a chevron divider, and the Requests and People items below it.
Each item is wrapped in a Radix Tooltip with its translated label.

The bottom panel is collapsible.
The user owns that state independently of the active view, so they can hide Requests and People even while one of them is open.

If the parent does not pass both create callbacks the rail renders a disabled `+` button instead of the CreateMenu, which keeps the layout slot balanced when the rail is used in chrome only contexts like Storybook or tests.

== Props

[cols="1,3", options="header"]
|===
| Prop | Description

| `onNewFolder` | Fired by the CreateMenu's "New folder" item
| `onFilesPicked` | Fired with the files the user picks via the CreateMenu's "Upload files" item
|===
