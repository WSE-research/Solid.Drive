= CreateMenu

Dropdown anchored on the NavRail's Create control.

* "New folder" fires `onNewFolder`.
* "Upload files" opens the OS file picker via a hidden input and forwards the chosen files to `onFilesPicked`.

The trigger renders as an icon-only `+` circle when `expanded` is false, or as a `+ Create or upload` pill when true.

== Props

[cols="1,3", options="header"]
|===
| Prop | Description

| `onNewFolder` | "New folder" handler.
| `onFilesPicked` | "Upload files" handler.
| `expanded` | When true, renders the pill trigger. Defaults to false.
| `labelShort` | Override for the collapsed accessible name. Defaults to the `oneDriveLayout.create` i18n string.
| `labelLong` | Override for the expanded label and accessible name. Defaults to `oneDriveLayout.createOrUpload`.
|===
