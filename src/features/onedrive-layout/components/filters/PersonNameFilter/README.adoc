= PersonNameFilter

Free text input that filters a list of contacts by display name or WebID.
Stateless.
The parent owns the value through link:../../../hooks/useSharedFilters/[useSharedFilters] or link:../../../hooks/useRecentFilters/[useRecentFilters].

== Props

[cols="1,3", options="header"]
|===
| Prop | Description

| `value` | Current query string
| `onChange` | Fired whenever the user edits the input
|===
