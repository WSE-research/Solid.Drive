= SharedFilters

Layout shell that pairs the `TypeFilterChips` chip row with the `PersonNameFilter` input.
Purely presentational.
All state comes from link:../../../hooks/useSharedFilters/[useSharedFilters] via the parent.

Scoped under the `<shared-filters>` element so the layout CSS can target it.

== Props

[cols="1,3", options="header"]
|===
| Prop | Description

| `chips` | Chip set derived from observed catalog entries
| `filters` | State and actions from link:../../../hooks/useSharedFilters/[useSharedFilters]
|===
