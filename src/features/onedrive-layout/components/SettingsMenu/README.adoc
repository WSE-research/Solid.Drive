= SettingsMenu

Gear-icon dropdown in the OneDrive topbar.
Hosts three Radix sections separated by dividers: the language radio group, the theme select, and the layout toggle.

Stateless composition.
The language section reads / writes through `react-i18next`; the theme section delegates to link:../ThemeToggle/[ThemeToggle]; the layout section delegates to link:../LayoutToggle/[LayoutToggle].
