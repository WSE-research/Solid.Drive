= RecentView

Home / Recent view for the OneDrive layout.

Reads the user's DCAT catalog, sorts entries by `modified` desc, and renders them in a flat Name / Opened / Owner table that mirrors the OneDrive Home recent section.

The inline toolbar above the table carries the type chips derived from the catalog's observed schema.org classes plus a person and name filter input.
At narrow viewports the chip row collapses into a dropdown.

All filter, search, and sort state lives in link:../../../hooks/useRecentFilters/[useRecentFilters].

== Internal pieces

* `RecentFilesTable` renders the flat Name / Opened / Owner rows.
