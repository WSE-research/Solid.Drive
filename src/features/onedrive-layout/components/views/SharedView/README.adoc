= SharedView

Shared view for the OneDrive layout.

Returns two sibling elements:

* `<shared-toolbar>` claims the OneDriveLayout `toolbar` grid area and replaces the page header title row.
* `<shared-body>` claims the `main` grid area.

Both siblings share the filter state from link:../../../hooks/useSharedFilters/[useSharedFilters], the active tab, and an observed types collector, so the chip set always reflects the catalog data of whichever tab is active.

Switching between With you and By you resets the chip selection and the observed types collector, so the other tab does not leak into the new one.

== Internal pieces

* `SharedToolbar` renders the tabs, the chip dropdown, and the person filter.
* `SharedSelectionToolbar` replaces the toolbar while a row is selected, exposing Open and Download plus a clearable selection count.
* `SharedBody` wraps `SharedFilesTable` per tab.
* `SharedFilesTable` renders the flat With you and By you rows.
* `useObservedSharedTypes` collects schema.org class observations from child tables and derives the chip set.
* `useSharedSelection` tracks the active row, toggles off when reselected, and clears on Escape.
* link:../../FilePreviewDialog/[FilePreviewDialog] is the in-app preview the Open action mounts; it reuses the existing `FileMediaPreview` from the file explorer.
