= features

UI features, each self contained in its own folder with components, hooks, and services.
Features may import from `@/infrastructure` and `@/shared`.
Cross feature imports are not allowed except that `file-explorer` imports from `sharing` (WAC hooks).

== Contents

link:auth/[auth/]:: Login/logout, provider selection, language switching
link:file-explorer/[file-explorer/]:: Browse Pod files, upload, download, share, navigate folders
link:profile/[profile/]:: View and edit user profile, manage contacts, handle access requests
link:sharing/[sharing/]:: Low level WAC ACL hooks; consumed by `file-explorer/SharePanel`
