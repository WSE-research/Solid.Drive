= LanguageSwitcher

A `<select>` dropdown for switching the UI language at runtime.
Uses `i18n.changeLanguage` from i18next.

== Features

[cols="1,3", options="header"]
|===
| Feature | Description

| Language options | Supported languages are defined in `SUPPORTED_LANGUAGES` in `@/config`
| Adding a language | Requires a locale file in `app/locales/` and an entry in the config array
|===
