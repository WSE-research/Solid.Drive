= Header

The site header. Manages Solid authentication UI for both login and logout states.

== States

[cols="1,3", options="header"]
|===
| State | Renders

| Logged out | Provider dropdown (`SOLID_PROVIDERS`), optional custom URL input, login button (disabled until a provider is selected), link to get a Pod, `InstallAppButton`
| Logged in | Display name (resolved via `vcard:fn` or `foaf:name`), logout button, language switcher, `InstallAppButton`
|===

== See also

* link:../../hooks/useAuth/[useAuth] is the hook this component uses to read authentication state and trigger login and logout.
