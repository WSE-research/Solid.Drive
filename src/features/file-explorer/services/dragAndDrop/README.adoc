= dragAndDrop

Pure helpers for drag-and-drop event inspection.

== Exports

[cols="1,3", options="header"]
|===
| Function | Behaviour

| `hasUnsupportedFolderDrop(dataTransfer)`
| Returns `true` when at least one of the dropped items is a directory. `FileExplorer` uses this to surface a dedicated "folders aren't supported" message before falling through to validation.
|===
