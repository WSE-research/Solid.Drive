= useDriveInitialization

Composes `usePodDiscovery` with URL backed folder navigation, so the file explorer can use browser history and shareable links.

== Features

Pod setup:: Delegates to `usePodDiscovery` for storage root discovery and app container creation.
Navigation state:: Keeps `currentUri` and `breadcrumbs` in sync with React Router via the `?folder=` search param. Invalid or missing values fall back to the storage root chosen at discovery time.
Storage retry:: Exposes `handleRetryStorage` from `usePodDiscovery` for both manual and auto retry.

The contact list used to live here too.
It was moved into a standalone link:../useContacts/[useContacts] hook so this one only deals with storage and navigation.

== Returns

The `usePodDiscovery` fields `appContainerUri`, `storageRootUri`, `noStorageDetected`, and `handleRetryStorage`, plus navigation helpers `currentUri`, `setCurrentUri`, `breadcrumbs`, `setBreadcrumbs`, `handleNavigate`, and `handleBreadcrumbClick`.
