= useDriveInitialization

Composes `usePodDiscovery` with URL-backed folder navigation so the file explorer can use browser history and shareable links.

== Features

Pod setup:: Delegates to `usePodDiscovery` for storage root discovery and app container creation
Navigation state:: Keeps `currentUri` and `breadcrumbs` in sync with React Router (`?folder=`); invalid or missing values fall back to the storage root chosen at discovery time
Contact list:: Reads the user's `foaf:knows` list and exposes it so `SharedWithMeSection` can load shared files
Storage retry:: Exposes `handleRetryStorage` (from `usePodDiscovery`) for both manual and auto-retry

== Returns

`usePodDiscovery` fields (`appContainerUri`, `storageRootUri`, etc.) plus navigation helpers: `currentUri`, `setCurrentUri`, `breadcrumbs`, `setBreadcrumbs`, `handleNavigate`, `handleBreadcrumbClick`.

`contacts`:: WebIDs from the user's `foaf:knows` list
