= file-explorer/hooks

React hooks for the file explorer feature.

== Contents

link:useAccessRequests/[useFileAccessRequests]:: Sends access requests to a contact's inbox for individual files or all files at once
link:useCreateFolder/[useCreateFolder]:: Validates folder names and creates new LDP BasicContainers on a Solid pod
link:useDriveInitialization/[useDriveInitialization]:: Discovers Pod storage, creates app container, sets up navigation state
link:useFilePreview/[useFilePreview]:: Creates a blob URL for previewing a binary resource; revokes on cleanup
link:useFileSharing/[useFileSharing]:: Checks if a resource has been shared (has non-owner ACL agents)
link:useFileTypes/[useFileTypes]:: Loads and caches file type definitions from `tbox.ttl`
link:useFileUpload/[useFileUpload]:: Executes the full upload sequence (container, binary, metadata, catalog)
link:useFileValidation/[useFileValidation]:: Validates upload form fields against SHACL shapes
link:useNavigation/[useNavigation]:: Manages current folder URI and breadcrumb trail
link:usePodDiscovery/[usePodDiscovery]:: Finds `pim:storage` in the user's profile
link:useSharedCatalog/[useSharedCatalog]:: Resolves shared files from a contact's Pod
