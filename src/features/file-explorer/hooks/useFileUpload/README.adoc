= useFileUpload

Handles the full file upload sequence to a Solid Pod.
Cleans up the created container on failure to avoid orphaned data.

== Upload sequence

. Creates a child container inside `mainContainer`
. Uploads the binary file
. Writes `index.ttl` with DCAT metadata (title, publisher, MIME type, size, upload date)
. Appends a dataset entry to `catalog.ttl`
. Links the catalog to the user's profile (if not already linked)

== Returns

`isUploading`:: `true` while any step of the upload sequence is in progress
`upload`:: Function that starts the upload sequence
