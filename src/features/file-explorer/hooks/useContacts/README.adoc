= useContacts

Reads the signed in user's Solid profile and returns the WebID list of the agents they `foaf:knows`.
Moved out of `useDriveInitialization` so the bootstrap hook only deals with storage and navigation.

Returns an empty array until the profile resolves.
The value is derived, so consumers can read it freely without further memoisation.

== Returns

`contactWebIds`:: `string[]`. WebIDs of the user's contacts, in profile order.
