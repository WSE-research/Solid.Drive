= useFilePreview

Loads a binary Solid resource and returns an object URL for in-browser preview.
Creates a blob URL with `URL.createObjectURL` and automatically revokes it on cleanup to prevent memory leaks.

== Returns

`previewUrl`:: Blob URL ready for use in `<img>`, `<video>`, or `<iframe>`, or `undefined` while loading

* link:../../components/FileCard/README.adoc[FileCard] is the component that calls this hook to render file previews.
