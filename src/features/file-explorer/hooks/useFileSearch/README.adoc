= useFileSearch

Debounced, case-insensitive substring filter over catalog entries.

== Behaviour

. The raw `query` is debounced by 200 ms.
. The debounced query is whitespace-split into terms; empty/whitespace-only input yields an empty result set.
. An entry matches only if every term is a substring of its `title` or `mediaType` (case-insensitive).

== Parameters

[cols="1,1,3", options="header"]
|===
| Parameter | Type | Description

| `entries`  | `CatalogEntry[]` | Entries to filter (normally from `useCatalog`)
| `query`    | `string`         | Raw user input
|===

== Returns

[cols="1,1,3", options="header"]
|===
| Key | Type | Description

| `debouncedQuery` | `string`         | The query actually applied (empty when input is blank)
| `results`        | `CatalogEntry[]` | Entries matching every term
|===
