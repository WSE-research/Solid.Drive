link:SharedWithMeSection/README.adoc[SharedWithMeSection]:: Shows files shared by contacts; reads per-contact shared catalogs and checks accessibility
link:TypeFolder/README.adoc[TypeFolder]:: Collapsible folder grouped by schema.org type, with request access buttons per-file
