= file-explorer

The main file management feature.
Lets users browse their Pod, upload files, download, share with contacts, and see files that contacts have shared with them.

link:services/fileFilter/[services/fileFilter]:: Determines which files are visible vs. hidden (system files, shared catalogs)

The feature is organized around the `FileExplorer` component, which coordinates navigation state and delegates rendering to the sub-components.
