= ProfileSidebar

Layout container for the left sidebar.
Assembles three panels in order and resolves the catalog URI from the user's profile to pass down to `RequestsPanel`.

Shown only when the user is logged in. The `App` component conditionally renders it.

== Layout

link:../ProfileCard/README.adoc[ProfileCard]:: Avatar + name with inline edit
link:../ContactsList/README.adoc[ContactsList]:: Add/remove contacts, view rejection notices
link:../RequestsPanel/README.adoc[RequestsPanel]:: Incoming access requests (only shown if catalog is available)
