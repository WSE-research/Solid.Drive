= ContactsList

Lists the user's contacts (from `foaf:knows`) and provides an input for adding new ones.
WebID input is validated to be a valid `http(s)://` URL before calling `addContact`.

== Features

Rejection notifications:: Loaded on mount from the owner's inbox and displayed in each `ContactRow`
Clearing rejections:: Removes the rejection from the local map; it is not deleted from the inbox until the user explicitly requests access again
