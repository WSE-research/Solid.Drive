= ProfileCard

Displays the logged-in user's avatar, display name, and WebID.
Entering edit mode lets the user change their display name and upload a new avatar.

== Features

View profile:: Shows avatar, display name, and WebID in read-only mode
Edit mode:: Toggled via "Edit Profile" button; snapshot-and-restore on cancel
Name editing:: Inline text input; saved via `useProfile.save`
Avatar upload:: File picker uploads to `{storageRoot}public/avatar.{ext}` via `useProfile.uploadAvatar`

== Hooks

link:../../hooks/useProfile/[useProfile]:: Owns name, imgUrl, displayName, isLoading, isUploadingAvatar, save, and uploadAvatar
link:../../../../shared/contexts/NotificationContext/[useNotifications]:: Shows success and error toasts after save and avatar upload
