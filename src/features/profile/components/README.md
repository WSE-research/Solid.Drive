# profile/components

## Overview

UI components for the profile sidebar.

## Contents

| Name | Description |
|---|---|
| **ProfileSidebar** | Layout wrapper that assembles `ProfileCard`, `ContactsList`, and `RequestsPanel` |
| **ProfileCard** | Shows avatar + display name; inline edit mode for name and avatar upload |
| **ContactsList** | Lists contacts with an input to add a new WebID; shows rejection notifications from inbox |
| **ContactRow** | One contact row with a request-access button and a remove button |
| **RequestsPanel** | Collapsible panel for incoming access requests; approve grants WAC access, deny sends a rejection notification |
