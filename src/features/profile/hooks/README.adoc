= profile/hooks

React hooks for profile and social data.

== Contents

link:useProfile/[useProfile]:: Read/write the logged-in user's `foaf:name` and `foaf:img` fields
link:useContacts/[useContacts]:: Read and modify the `foaf:knows` contact list
link:useContactRequest/[useContactRequest]:: Per-contact access-request lifecycle: discover inbox, post the catalog access request, retry after rejection
link:useAccessRequests/[useAccessRequests]:: Load, approve, and deny access requests from the user's LDP inbox
link:useRequesterProfile/[useRequesterProfile]:: Resolve a requester's Solid profile (display name, avatar, loading state) from a WebID
link:useSeenRequests/[useSeenRequests]:: Persist which inbox request message URIs have been viewed
