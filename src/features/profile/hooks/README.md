# profile/hooks

## Overview

React hooks for profile and social data.

## Contents

| Name | Description |
|---|---|
| **useProfile** | Read/write the logged-in user's `foaf:name` and `foaf:img` fields |
| **useContacts** | Read and modify the `foaf:knows` contact list |
| **useAccessRequests** | Load, approve, and deny access requests from the user's LDP inbox |
