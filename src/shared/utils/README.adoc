= shared/utils

Pure utility functions. No React, no side effects.

== Barrel exports (`@/shared/utils`)

`formatBytes(bytes)`:: Formats a byte count to `"1.2 KB"` / `"3.4 MB"`.
`getInitial(name)`:: First character of a name, uppercased.
`getProfileDisplayName(profile, webId)`:: Best display name from a Solid profile.
`getWebIdFallbackName(webId)`:: WebID-derived fallback name (subdomain or path segment).
`isAbsoluteUri(value)`:: `true` if the value starts with `http://` or `https://`.
`truncateDisplayName(name)`:: Shortens a name to `MAX_DISPLAY_NAME_LENGTH`, appending an ellipsis.
