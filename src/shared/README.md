# shared

## Overview

Reusable building blocks used across features. Nothing here should depend on a specific feature.

## Contents

| Name | Description |
|---|---|
| **components/** | Generic UI components (`Avatar`, `ErrorBoundary`, `Toast`) |
| **contexts/** | React contexts (`NotificationContext` for toasts and confirm dialogs) |
| **hooks/** | Shared React hooks used across features (`useContactRejections`) |
| **utils/** | Pure helper functions (`formatBytes`, `getInitial`, `getProfileDisplayName`, `isAbsoluteUri`) |
