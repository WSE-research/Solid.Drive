= ClassicLayout

The original two pane shell.
Profile sidebar on the left, file explorer on the right.
Pulled out of `App.tsx` so the root component can swap between this and the OneDrive layout based on the user's preference.

Stateless and self contained.
Mounts `ProfileSidebar` and `FileExplorer` and lets them own their own data.
