= App

Root component that bootstraps the application.
Wraps everything in the Solid authentication provider and notification provider, then renders the header and main layout.

== Layout

[cols="1,2", options="header"]
|===
| Condition | Renders

| Always | `<BrowserSolidLdoProvider>` wrapping `<NotificationProvider>` wrapping `<Header />`
| Logged in | `<ProfileSidebar />` and `<FileExplorer />`
| Logged out | `<FileExplorer />` only
|===
