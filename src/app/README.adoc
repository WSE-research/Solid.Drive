= app

Application bootstrap and setup.
Contains the root React component, the layout shells, i18n configuration, locale translation files, and the bootstrap level hooks that span both layouts.

== Contents

[cols="1,3", options="header"]
|===
| Name | Description

| *App/* | The root React component. Wires together providers and picks a layout based on the persisted preference.
| *AuthCallbackSkeleton/* | Skeleton placeholder shown while the Solid auth provider is processing an OIDC redirect.
| *ClassicLayout/* | The original two pane shell with the profile sidebar and the file explorer.
| *hooks/* | Bootstrap level React hooks. Currently just `useSessionContinuity`.
| *i18n/* | i18next configuration: language detection, supported languages, and fallback.
| *locales/* | Translation JSON files, `en.json` and `de.json`.
|===

The OneDrive alternate shell lives under link:../features/onedrive-layout/[features/onedrive-layout/] and is mounted by `App` when the user opts in via the layout toggle.

`main.tsx` renders `<App />` into the DOM.
Everything else flows from there.
