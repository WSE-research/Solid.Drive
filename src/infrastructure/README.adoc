= infrastructure

Protocol and service layer — no React, no UI.
This code talks to the network, parses RDF, and handles Solid-specific protocols.

Infrastructure modules only depend on `@/config`, `@/types`, and each other.
They never import from `@/features` or `@/shared/contexts`.
