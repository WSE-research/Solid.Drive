= wac

Web Access Control (WAC) implementation for Solid resources.
WAC is the Solid authorization protocol where permissions are stored in `.acl` files alongside resources.

The link:../../features/sharing/hooks/useAclManager/[useAclManager] hook in `features/sharing` provides the React interface on top of this module.
