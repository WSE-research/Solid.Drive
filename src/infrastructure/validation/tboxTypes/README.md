# tboxTypes

## Overview

Type definitions for TBox validation.

## Contents

| Type | Description |
|---|---|
| `PropertyConstraint` | Constraint for a single property in a SHACL shape |
| `ShapeDefinition` | Definition of a SHACL NodeShape with required and optional properties |
| `ValidationResult` | Result of validating metadata against a shape |
| `PropertyViolation` | A missing required property from validation |
