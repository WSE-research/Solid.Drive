= validateFile

Pure async validator for `File` objects. Used by both the upload form (via the wrapping `useFileValidation` hook) and the bulk-drop pipeline (`useUploadQueue`).

== Behaviour

. Loads the TBox via `loadTBox` (cached at the validator module level).
. Resolves the file's class URI from its MIME type via `resolveClass`.
. Builds a metadata snapshot equivalent to what the existing upload form derives.
. Returns the `ValidationResult` from `validateMetadata`.

== Parameters

[cols="1,1,3", options="header"]
|===
| Parameter     | Type              | Description

| `file`        | `File`            | File to validate
| `title`       | `string`          | User-provided title; falls back to `file.name` when empty
| `description` | `string`          | Optional description
| `webId`       | `string`          | Publisher WebID for the snapshot
|===
