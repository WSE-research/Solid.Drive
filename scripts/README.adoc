= scripts

Build time and development utility scripts.
Run with Node.js directly.

== Contents

`extract-tbox.mjs`::
Downloads the SHACL TBox from datashapes.org and writes it to `public/tbox.ttl`.
+
----
node scripts/extract-tbox.mjs
node scripts/extract-tbox.mjs --source https://datashapes.org/schema.ttl
node scripts/extract-tbox.mjs --output public/tbox.ttl
----
+
The generated file carries a DO NOT EDIT header — re-run the script to refresh it.

`fix-ldo-types.mjs`::
Patches auto-generated LDO files in `src/.ldo/` to use `import type` instead of `import`.
Required after running `@ldo/cli` because the generator emits value imports where TypeScript's `--isolatedModules` expects type-only imports.
+
----
node scripts/fix-ldo-types.mjs
----

`generate-pwa-icons.mjs`::
Generates versioned PWA and Apple touch icons from `src/assets/solid-drive-logo.png` using `sharp` and writes them to `public/icons/`.
Produces five files: `pwa-192x192-v2.png`, `pwa-512x512-v2.png`, `pwa-512x512-v3.png`, `maskable-512x512-v2.png`, and `apple-touch-icon-180x180-v2.png`.
+
----
npm run pwa:icons
node scripts/generate-pwa-icons.mjs
----
+
Transparent icons use a fully transparent background; maskable and dark-background icons use `#07073800` (near-black).
Re-run whenever the source logo changes.
